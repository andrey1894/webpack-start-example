# WEBPACK START EXAMPLE

Install `yarn` package manager.

Install and build this app with:
``` text
    npm version: 6.14.12
    node version: 14.18.0
```

## Install packages

``` text
    yarn
```

### Run app local

``` text
    yarn start
    And open https://localhost
```

### Build app dev

``` text
    yarn build:dev
```

### Run app dev

``` text
    yarn build:dev:watch
```
### Build app prod

``` text
    yarn build:prod
```

### Run app prod

``` text
    yarn build:prod:watch
```
